2. Infrastructure: uv

2.1 Introduction to uv

uv is an extremely fast Python package installer and resolver, written in Rust. It serves as a high-performance replacement for pip and pip-tools.

Installation (via curl):

# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

3. Standards: Ruff

3.1 Introduction to Ruff

Ruff is a blazing fast Python Linter and Formatter. It replaces Flake8, isort, and Black, but runs 10~100x faster.

3.2 pyproject.toml Example

Modern Python projects use pyproject.toml for configuration. Here is a typical setup:

[build-system]
requires = ["setuptools>=61.0", "wheel"]
build-backend = "setuptools.build_meta"

[project]
name = "my_project"
version = "0.1.0"
description = "A modern Python project"
dependencies = []
requires-python = ">=3.10"

4. Experience: Rich

4.1 Terminal Enhancements

Rich makes terminal output beautiful. It supports colors, tables, progress bars, and Markdown rendering.

Example (Better Tracebacks):

from rich.traceback import install
install()

def faulty_function():
    result = 1 / 0
    return result

faulty_function()

With Rich, you'll get a clear, detailed error report with variable values and code context.

5. Conclusion

Combining VS Code/Cursor + uv + Ruff + Rich provides a fast, standardized environment for modern Python development.
