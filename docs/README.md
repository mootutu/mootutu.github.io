## 仓库结构速览

```
mootutu.github.io/
├── _layouts/          # 页面布局模板
├── _includes/         # 可复用组件
├── _data/             # 站点数据文件
├── _templates/        # 内容创建模板
├── assets/            # 静态资源（图片、CSS、JS）
├── blog/              # 英文博客内容
├── learning/          # 英文学习内容
├── cn/                # 中文版本（镜像结构）
├── docs/              # 维护文档（本目录）
├── _config.yml        # Jekyll 配置
├── index.html         # 首页
├── blog.html          # 博客列表页
└── learning.html      # 学习主题列表页
```
