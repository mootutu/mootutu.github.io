2. 底层：uv

2.1 uv 简介

uv 是一个速度极快的 Python 包安装器和解析器，由 Ruff 的作者 Charlie Marsh 开发。它可以看作是 pip 和 pip-tools 的高性能替代品，旨在彻底解决 Python 包管理的速度问题。

安装方法（推荐使用 curl）：

# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

3. 规范：Ruff

3.1 Ruff 简介

Ruff 是一个极速的 Python Linter 和 Formatter，用 Rust 编写。它几乎涵盖了 Flake8、isort、Black 等工具的所有功能，但速度快了 10~100 倍。

3.2 pyproject.toml 配置示例

现代 Python 项目推荐使用 pyproject.toml 进行统一配置。以下是一个集成了 Ruff、Pyright 等工具的典型配置：

[build-system]
requires = ["setuptools>=61.0", "wheel"]
build-backend = "setuptools.build_meta"

[project]
name = "my_project"
version = "0.1.0"
description = "A modern Python project"
dependencies = []
requires-python = ">=3.10"

# ==============================
# 🛠️ Ruff 配置（Lint + Format）
# ==============================
[tool.ruff]
preview = true
target-version = "py310"
fix = true
line-length = 88

[tool.ruff.lint]
select = [
  "E", "W", "F", "I", "UP", "C4", "TID", "ARG", "PTH"
]
ignore = ["E501", "D"]

[tool.ruff.format]
quote-style = 'single'
indent-width = 4

4. 体验：Rich

4.1 终端美化与增强

Rich 能够极大地改善命令行应用的开发体验。它可以让你的终端输出具有高亮颜色、表格、进度条甚至 Markdown 渲染。

安装与使用：

pip install rich

示例代码（增强错误报告）：

from rich.traceback import install
install()

def faulty_function():
    result = 1 / 0
    return result

faulty_function()

运行上述代码后，你将看到一个清晰、带变量值和代码上下文的错误报告，极大地缩短了调试时间。

5. 总结

通过组合使用 VS Code/Cursor + uv + Ruff + Rich，我们可以构建一个既快速又标准化的现代 Python 开发环境。这些工具不仅提高了代码质量，更重要的是在日常开发、安装依赖和调试问题时节省了大量时间。
