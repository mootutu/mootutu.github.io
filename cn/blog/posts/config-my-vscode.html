这篇文章记录我目前在 VS Code 上的个性化配置流程，包括毛玻璃、动画以及自定义开屏图标。步骤不多，但能让编辑器更有氛围感。

1. 开启毛玻璃效果

先安装插件 Vibrancy Continued，然后：

  • 按 Cmd + Shift + P 打开命令面板
  • 输入 Enable Vibrancy 并回车

2. 添加动画效果

安装 VSCode Animations 和 Custom CSS and JS Loader，再执行：

  • 按 Cmd + Shift + P
  • 输入 Enable Animations 并回车

3. 自定义开屏图标

我用的是 这个图标，下载后用 图片转 Base64 工具 转码。然后新建一个 user.css 文件，内容如下：

.editor-group-watermark .letterpress {
  background-image: url("base64放这里") !important;
  opacity: .75;
  aspect-ratio: 3 / 2 !important;
}

接着在 VS Code 的 settings.json 里添加：

"vscode_custom_css.imports": [
  "file:///你的user.css文件的绝对路径/user.css"
]

4. 启用自定义 CSS

最后在命令面板输入 Enable Custom CSS and JS 并回车，开屏图标就会生效。
